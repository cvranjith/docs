Create Message Bridge

Created by Ranjith Vijayan on Jul 17, 2020



Click on New and input information as below:

Select Source Bridge as existing Source Destination


Select Target Bridge Destination as existing Target Destination

As the destination is of WS hence the Message Provider would be selected as Other JMS


After that, we will click on Finish on next screen and back to the main menu, we will see message bridge created as below:

Click on Local2MQ to edit some parameters:
QOS Degradation Allowed to be ticked
