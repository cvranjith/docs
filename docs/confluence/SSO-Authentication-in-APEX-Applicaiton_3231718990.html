SSO Authentication in APEX Applicaiton

Created by Ranjith Vijayan on Sep 09, 2021

Applications hosted in apex.oraclecorp.com can be easily configured to with Oracle SSO for user authentication

New Applications

  • Create an application by clicking New Application
  • Under Settings → Authentication use "SAML Sign-in"
  • Your applicaiton will now work with login.oracle.com authentication 



Existing Applicaitons

  • Go to Shared Components → Authentication Schemes
  • Add a new Authentication Scheme by clicking "Create" button
  • Choose "Based on a pre-configured scheme from the gallery" and click Next
  • Enter a name and choose "SAML Sign In". Then click "Create Authentication Scheme" button.
  • Make sure the default authentication scheme is changed to the new scheme you created.
  • Optionally Delete the old authentication scheme
