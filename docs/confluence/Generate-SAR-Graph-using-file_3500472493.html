2. How To's

Generate SAR Graph using file

Created by Sougata Laga, last modified on Feb 02, 2022

Given a list of SAR reports (in .txt format) : 



  • Visit this link to download a zip file from github : https://github.com/UnixArena/UnixArena/blob/main/kSar-5.0.6.zip 



click the download button to start downloading.


  • Unzip the zip file, to find the contents of it as :



  • Go to command prompt in the laptop and give the command : java -jar kSar.jar



  • The following kSar interface opens up : 



  • Go to options in top left side and tick all the checkboxes :



  • In top left in 'Data' option -> click 'load from text file' :



  • Select the respective text file from the location :



  • Data import has been done :



  • In the left side, go to 'CPU' to see the generated graph : 



  • in top left, click Data → Export as jpg : 



  • give the respective JPG file name : 



  • In the final step, click on OK in the prompted box : 



  • Graph is generated for respective file. Repeat these steps for other files also if required.

That's it. Hope this helps!
